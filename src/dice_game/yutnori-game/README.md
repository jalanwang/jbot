# [하위] 윳놀이 보드 엔진 초기 구현 및 ROS2 연동

이 문서는 윳놀이 게임의 핵심 로직을 코드로 구현하고, 주사위 인식 시스템과 데이터를 주고받을 수 있는 환경을 구축하는 태스크를 관리한다.

---

## 1. 실행 가이드 (To-do)

### 배경
- 장기 게임 브랜치에서 윳놀이 브랜치로 전환이 완료되었으므로, 깨끗한 환경에서 윳놀이 전용 패키지를 구성한다.
- 주사위 인식 결과값($1 \sim 6$)이 들어왔을 때 윳판 위의 말이 실제로 움직이는 기능을 구현한다.
